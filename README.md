#  Excel Data Analysis Project

## Project Overview
This project analyzes data using Microsoft Excel to generate insights and visualize key trends. The analysis includes data cleaning, calculations, and interactive dashboards.

##  Objective
The goal of this project is to:
- Analyze data using Excel
- Create meaningful insights
- Build charts and dashboards for better understanding


##  Features
- Data cleaning and formatting
- Pivot Tables for analysis
- Charts and graphs for visualization
- Dashboard for data insights

##  Tools Used
- Microsoft Excel
- Pivot Tables
- Charts and Data Visualization

##  Outcome
The project demonstrates how Excel can be used to analyze data and present insights through visual dashboards.
